# アプリの説明

Haiku D&E とは？ (Develop ＆ Execute：俳句を創作し、このアプリを実行) 
　　　　[Q & A はこちら](#a1) 

あなたの大切な俳句を保存して、いつでも楽しめるアプリです

句帳に眠っている名句を、句会の結果を、どうぞ登録してください

俳句を作らない方も、好きな名句や忘れたくない句を保存して楽しんでください

お気に入りの俳句を登録保存して、句の日々の暗唱　や　評価の星をつけるなどして、将来の句集作成に役立てば幸いです

Q2. どうやって俳句を追加しますか？  <a id="A2"></a>


<img width=200 src=https://github.com/nakata-y/Haiku/assets/101084603/9b59a56d-f203-4cd5-b4a1-c8f5a18d83e2.jpeg>



A2. メイン画面右上の「+ 俳句追加」ボタンをタップして、入力画面で入力します。

　　俳句の上の句、下の句、中の句と分けて入力ください。

   そのほかの項目は、詳しくは、入力画面の各項目の「？」を押すと、説明が出てきます。
   
   または、下記の詳しい説明を参照ください。

   入力が終わったら、「新規追加追加する」ボタンを押してください。

   すでに登録されている俳句の最後に追加登録されます。
   
<img width=200 src=https://github.com/nakata-y/Haiku/assets/101084603/f7e3ad22-933f-41a6-b48e-95707b468186.jpeg>

Q3. 登録した俳句の並び順を変更できますか？   <a id="A3"></a>

A3. 移動したい俳句をタッチして押さえたままにして、持ち上がるまで待ちます。
別の場所にドラッグします。
ドラッグ中、俳句をドロップできる場所に　緑の＋マークが表示されます。
移動したいところで、手を離します。

参照）　iPadでドラッグ＆ドロップして項目を移動およびコピーする
https://support.apple.com/ja-jp/guide/ipad/ipadaa83b207/ipados

うまくいかない時は、、検索・並び替え・絞り込み　のページ（Q６参照）の右上の「編集」を押して、順番の変更や削除も可能です。


Q4. 登録した内容の保存はできますか？   <a id="A4"></a>
　　システム障害等でデータが消えたり、他のアプリで利用したいときのために。
  
A4. はい、テキストデータで保存できます。
　　　俳句だけの保存なら、メイン画面の右上の共有ボタンで、他のアプリに保存すれば安心です。
   　俳句以外の覚書等も保存したいときは、検索・並び替え・絞り込み　のページ（Q６参照）の右上に共有ボタンで、保存すれば安心です。
 

Q5. 友達と共有やフォトライブラリへの保存は可能ですか？   <a id="A5"></a>

A5. はい、メイン画面や検索・並び替え・絞り込み　のページ（Q５参照）の右上に共有ボタンで、
　　　他のユーザーと共有できます。
    また、右上のカメラマークをタップすると、フォトライブラリへの保存も可能です。

Q7. 手書き入力はどうやって使いますか？　   <a id="A7"></a>

A7. 画面下部にある「手書き入力」をタップすると、手書きでの俳句の入力や編集ができます。
　　Apple Pencilや指で手書きができます。
　　「Save」を押すと、フォトライブラリへの保存も可能です。

Q8. 句会休会のときの欠席投句にこのアプリは使えますか？　   <a id="A8"></a>

A8. いろいろ方法がありますが、例えば、
　　* 欠席者がこのアプリに　投句を登録。
   * 検索・並び替え・絞り込み　のページ の右上 の縦書きボタンを押す
   * 画面右下のカメラマークをタップして、フォトライブラリへの保存。
   * 上記をメールやLineで出席者へ送付。
   * 出席者は、２枚印刷して、１枚は投句控え。もう１枚は、切って短冊に。
      書写しの間違えも無しで、手間も要らずです。

 Q12. 俳句の作りませんが、このアプリで楽しめますか？　   <a id="A12"></a>

　A11.  勿論です！
　　　　俳句を作らない方も、好きな名句や忘れたくない句を保存して、
    
　　　　例えば散歩しながら、アプリをチェックし、名句を記憶すれば、
    
　　　　記憶力アップ！（ただ歩くより足し算をしたり、何か覚えながら歩く方がいいそうです）

##アプリ-俳句-D&Eの詳しい説明



＊ 保存した俳句を縦書きで、短冊に手書きしたように、一覧表示します。

* ご自分で俳句を作らない方は、お好きな俳句を登録して、覚書に作者名を入力するなどして、楽しんでください。

＊ 俳句を長押しして、好きな位置への移動が可能。

＊ 俳句の簡単な共有やフォトライブラリへの保存し、 他のユーザーやSNSへ共有ができます。
 
＊ 短冊風・色紙風のデザイン表示。

＊ 詳細検索、並び替え、絞り込み機能があります。

＊ 俳句の追加や編集が簡単。

＊ 手書き入力機能で、吟行中のメモや、ふと浮かんだ時の下書き可能。

＊ ヘルプページへの簡単なアクセス。「？」ボタン。

--------------------------------------------------------------
ページ下部の３つのアイコンをタップして、3つの画面は切替わります　（タブの切り替え）
--------------------------------------------------------------
1. メイン画面は、保存されている俳句の一覧。

2. 検索・並び替え・絞り込み
   
保存されている俳句を検索したり、並び替えたり、特定の条件で絞り込むことができます。

3. 手書き入力:
   
「手書入力」アイコンをタップすると、手書きの入力画面。
　　
  吟行中のメモや、ふと浮かんだ時の下書きなどに利用ください。

---------
さらに詳細に
---------

1. メイン画面は、
   
* 俳句の登録・追加
  
    * 画面右上にある「+　俳句追加」というボタンをタップしてください。
      
    * 入力画面が出てきたら、俳句の上の句、中の句、下の句を入力してください。
  
　　　　また、覚書、年月日、星の数、任意番号などの詳細情報も追加できます。  
    
      俳句入力：「上の句」、「中の句」、「下の句」

     　＊＊＊ 以下の覚書・年月日・星の数・任意番号は、全て任意です。
  
      覚書　　：何か記録しておきたいものを、入力ください。入力は必須ではありません。
      
      　　　　　（例）句会の名前や、句会での評価等。また、自分の句でないときは、作者名など自由に入力ください。

      年月日　：選択しなければ、アプリに登録した年月日です。年月日を俳句の作成日等に変更したい場合は、選択してください。
  
      星の数：評価したい星をタップしてください。タップされた星の数が保存されます。  指定しない場合は、星の数は０となります。

      
      任意番号：独自に割り当てたい任意の番号があれば入力ください。入力は必須ではありません。
      
      　　　　　（例）登録順を付番したい、句集に入れる順番を付番したい時など入力してください。
           

* 入力が完了し画面上部の　青い「登録する」ボタンを押したら、追加することができます。
    
* 俳句の並び替え
  
    俳句を長押ししたまま、ドラッグアンドドロップすることで、表示の順番を変更することができます。
      
* ある一句の閲覧・編集・削除
  
    俳句をタップすると、その俳句の詳細情報が表示されます。
      
    画面上部に、3つの選択肢（俳句のみ表示、全て表示、色紙）があり、画面の表示内容が変わります。

  　色紙を選び名前を入力すれば、色紙に名前も入ります。

  　　（注意）登録の際、上の句、中の句、下の句に分けたとおり、色紙に３行分かれて表示します。

      　      
    フォトライブラリへの保存

      　画面の右上の写真アイコンのボタンがあり、これをタップすると現在の画面をフォトライブラリへの保存できます。
      
    内容の変更
      
      　上部の「内容の変更 」　ボタンを押したら、　変更画面が表示。 変更したい部分を修正して、「変更」ボタンを押したら、変更されます。

       　「x」ボタンを押したら、変更を保存せずに閉じます。


  　登録句の削除
    
        「ゴミ箱」ボタンをクリックしたら、編集中の俳句をアプリから完全に削除することができます。

  

2. 検索・並び替え・絞り込み:
   
   　俳句のほかに、覚書、日付、星の数、任意の番号も、横書きで表示されます。

   　編集（削除・移動）：　右上の編集ボタンを押してください。

　　　検索バーを利用して俳句を検索したり、表示の並び替えをしたり、特定の条件で絞り込みを行うことができます。

    検索や絞り込みの条件に合わせてソートされた俳句の一覧を表示します。
      
    並替の条件の選択項目が、表示され、複数のソートオプションから選ぶことができます。
      
    「絞り込み」と表示されたボタンをタップすると、詳細な絞り込みの画面へ。
      
    もし既に絞り込みの条件が設定されている場合、その条件に応じて「期間 絞込中」、「星の数 絞込中」、「任意番号 絞込中」といった黄色のラベルが表示されます。
    
    条件をクリアするしたいときは「x」ボタン。
      
    並び替え・絞り込みをしているときは、右上の編集ボタンを押しても、編集（削除・移動）できません。編集をしたい場合は、並び替え・絞り込みをやめてください。
    
* 右上のボタン
  
    縦書き表示に変更
      
    テキストデータの共有　（他のアプリや友人へ　共有可能です）
      
* 絞り込みの設定画面

    期間指定: 期間を指定するを選んだ場合、開始日と終了日のカレンダーから日付を選択することができます。

  　 　星の数を指定: 星の数を0から3までの間で選択することができます。   
   
  　 任意番号指定: 2つの数字を任意で入力することができます。
   
3. 手書き入力:
   
Apple Pencilや指を使っての手書きのメモができます。

* Showボタン：ツールの選択: ペン、鉛筆、消しゴムなどの描画ツールを選択・切り替えることができます。
  
* Clearボタン：クリア: 一度描かれた内容を全てクリアすることができます。
